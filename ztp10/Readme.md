# Trabajo practico 10
## Modos
Tocando las siguientes teclas, se puede entrar a los distintos modos:
* q: salir del programa.
* d: dibujar.
* r: borrar dibujo.
* n: Cambia aruco por nave. Si se apreta nuevamente, desaparece la nave.
* a: Cambia el aruco por un video. Para salir de este modo, es necesario apretar la tecla q.
* g: Ejecuta un juego en el cual hay que esquivar obstaculos. q para salir.
